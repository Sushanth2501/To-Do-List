# 📝 Java To-Do List (Console App)

A simple console-based To-Do List application written in Java.  
It allows users to:

- ✅ Add tasks  
- 📋 View tasks  
- ❌ Remove tasks  
- 🔚 Exit the application

---

## 💻 How to Run

1. Open terminal or command prompt
2. Compile the Java file:
   ```bash
   javac ToDoListApp.java
